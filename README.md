# Hearthbag

This addon provides a button that can be used with a variety of hearthstones including a unique skin made of the Hearthstone game cardback textures.

### Commands

Commands can be accessed via `/hearthbag` or `/hb`. Subcommands include:
 - /hb combat
   - Toggle the moveable combat frame
 - /hb anchor
   - Set the Hearthbag Parent to the frame under your mouse or to the specified frame name (BagItemSearchBox, Minimap, PlayerFrame)
 - /hb reset
   - Reset to your Hearthbag to centered position


### Limitations

The Hearthbag frame won't be displayed on certain frames during combat due to limitations, as the frame will become "Protected". The Combat Frame is an optional frame for Hearthbag to still remain accessible during combat.

### Missing Hearthstones

I may be a bit slow to update some of the current missing hearthstones and add skins for them. I'll eventually get to them, but keep in mind they're all manually done by hand in my free time.
